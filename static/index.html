JVM Tuned for Gaming

Vanta ships with carefully tuned G1GC flags, aggressive memory pre-touch, and parallel reference processing — the same flags used by competitive PvP clients and large-scale servers.

The RAM slider (2–16 GB) instantly adjusts both -Xmx and -Xms with matching values to eliminate GC pauses caused by heap resizing.
